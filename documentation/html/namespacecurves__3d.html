enum class curves_3d::curve_3d_error : std::uint8_t
strong

Error codes used by curve factory methods.

All static create(...) methods validate input parameters and return an error code in std::unexpected(...) if validation fails.

Enumerator
bad_radius_circle_3d 

Invalid circle radius (must be strictly positive).

bad_radii_ellipse_3d 

Invalid ellipse radii (intended: both radii must be strictly positive).

bad_helix_3d 

Invalid helix parameters (radius and pitch must be strictly positive).
