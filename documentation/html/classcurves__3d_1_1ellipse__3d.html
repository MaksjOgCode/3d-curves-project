Detailed Description

Ellipse in the XY plane centered at the origin.

Parametric form:

\[ x(t)=R_x\cos(t), \quad y(t)=R_y\sin(t), \quad z(t)=0 \]

First derivative:

\[ x'(t)=-R_x\sin(t), \quad y'(t)=R_y\cos(t), \quad z'(t)=0 \]

Note
The object is disabled when rx_ <= 0 && ry_ <= 0.
